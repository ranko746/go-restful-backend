# go-restful-api
Go MongoDB Restful API

### You can clone this repo using 
```bash
git clone "https://github.com/AlxPatidar/go-restful-api" go-rest-api
```

### Change directory
```bash
cd go-rest-api
```
### Copy env and update env variable
```bash
cp config/.env.example config/.env
```

### Now last step run
```bash
go run main.go
```
